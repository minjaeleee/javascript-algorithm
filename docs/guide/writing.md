# 학습 노트 작성법

## 개념 문서

개념 문서는 다음 구조를 기본으로 사용합니다.

1. 학습 목표
2. 핵심 개념
3. 동작 원리 또는 시각적 예시
4. JavaScript 구현
5. 시간 복잡도와 공간 복잡도
6. 자주 하는 실수와 경계 조건
7. 짧은 예제
8. 복습 질문

## 문제 풀이 문서

문제 풀이 문서는 문제 요약, 제약 조건, 접근 방법, 알고리즘 선택 근거, JavaScript 풀이, 복잡도 분석, 개선점과 배운 점을 순서대로 기록합니다.

> 강의 문장을 그대로 옮기기보다 직접 이해한 표현으로 요약합니다. 외부 자료를 사용하면 출처와 링크를 표시합니다.
