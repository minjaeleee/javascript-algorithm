# JavaScript 핵심 자료구조 알아보기

자료구조는 데이터를 어떤 형태로 저장하고 다룰지 결정하는 방법이다. 같은 문제라도 자료구조를 어떻게 선택하느냐에 따라 실행 시간과 메모리 사용량이 크게 달라진다.

이 챕터에서는 자료구조를 평가하는 기준부터 배열, 연결 리스트, 스택, 큐, 트리, 우선순위 큐와 그래프 표현 방법까지 살펴본다.

## 학습 목표

- 시간 복잡도와 공간 복잡도로 자료구조의 성능을 비교한다.
- 선형 자료구조와 비선형 자료구조의 차이를 설명한다.
- JavaScript에서 스택과 큐를 효율적으로 구현한다.
- 트리, 힙, 그래프의 기본 용어와 표현 방식을 이해한다.
- 문제의 연산 조건에 맞는 자료구조를 선택한다.
