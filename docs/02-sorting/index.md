# JavaScript 정렬(Sorting) 알고리즘

정렬은 데이터를 특정 기준에 따라 순서대로 나열하는 알고리즘이다. 숫자를 오름차순으로 정렬하거나, 문자열을 사전순으로 정렬하거나, 객체 배열을 특정 속성 기준으로 정렬할 때 사용한다.

정렬 알고리즘은 코딩 테스트에서 매우 자주 등장한다. 직접 정렬 알고리즘을 구현해야 하는 경우도 있고, JavaScript의 정렬 라이브러리를 정확히 이해하고 사용해야 하는 경우도 있다.
